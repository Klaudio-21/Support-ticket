# Support-ticket

Support ticket application built with the MERN stack.

# What was used in this  project?

- React Js
- Redux
- Node Js
- Express Js
- MongoDB
- Mongoose

### Install backend dependencies

```bash
npm install
```

### Install client dependencies

```bash
cd frontend
npm install
```

### Run app in development (frontend & backend)

```bash
npm run dev
```
